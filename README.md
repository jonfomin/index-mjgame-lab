# index-mjgame-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以实验、原型或静态展示为主要目的，不针对任何特定域名或网站。仓库内容以纯前端静态文件为主，方便直接浏览或嵌入。

## 目录说明

- `/` — 根目录，存放主要 HTML 页面及相关资源（CSS、JS、图片等）
- 每个子文件夹（若有）代表一组相关页面或独立实验单元
- 文件命名尽量保持清晰，便于识别内容主题

## 页面归档说明

本仓库中的所有 HTML 页面均为独立归档，彼此之间无强制依赖关系。各页面功能、样式和逻辑均在其自身文件内实现，部分页面可能引用外部公共库（如通过 CDN 加载）。建议使用现代浏览器打开以获得最佳显示效果。

## 维护说明

- 本仓库不定期更新，新增或修改页面均会记录在提交历史中
- 不保证所有页面在未来版本中保持完全兼容
- 欢迎通过 Issues 提交反馈或建议，但请注意本仓库不针对具体业务或服务
- 若需引用或分发本仓库内容，请遵守仓库所附的开源许可协议
